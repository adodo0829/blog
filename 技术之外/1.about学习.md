# 谈一下个人学习
上了这么多年的学, 我想每个人都形成了一套自己的学习方式,这里我也总结一下自己的. 当然是关于工作后的学习...

## 学习一门语言
```
- 语法,词法
- 语句
- 语句组合
- 应用
每个语法有其对的规则, 我们需要遵守, 不然就要踩坑
编程更多的是学习其思想, 对业务逻辑进行抽象提炼, 用语句来实现的一个过程
```

## 学习源码
```
针对性的学习, 不建议从头看到尾;

可以从某个功能入手, 梳理其工作流程,周边依赖
```
## 学习一个知识点
遵循下面的规则
```
是什么: 如何定义, 背景
怎么做: 如何使用,
怎么实现: 内部运行机制
注意点: 需要注意的事项
应用场景: 相信存在即必然, 适用哪些地方
```
