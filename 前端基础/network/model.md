# 网络层

## OSI七层模型

应用
  应用层: 用户接口
  表示层: 描述数据
  会话层: 建立管理回话     应用层数据(报文)

传输层: 数据传递给对方     传输层头部,应用层数据 数据段(端口)

网络层: 寻址                                数据包(IP)
- 路由器

数据链路层: 连接设备
- 交换机            

物理层: 0, 1数据传输
- 光纤

## 七层模型功能

报文: 应用层 + 数据
数据段: 传输层 + 数据 + 端口
数据包: 网络层 + 数据 + + 端口 + ip地址
数据帧: 链路层 + 数据 + 端口 + ip + mac地址
